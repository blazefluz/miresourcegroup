================================================================================
                           TROUBLESHOOTING
================================================================================

Build Fails:
  - Verify all environment variables are set
  - Check Sanity credentials are correct
  - Run 'npm run build' locally first
  - Check Vercel build logs for details

Images Not Loading:
  - Verify SANITY_API_TOKEN has read permissions
  - Check Sanity project ID is correct
  - Verify images exist in Sanity Studio
  - Check browser console for CORS errors

Studio Not Working:
  - Add Vercel domain to Sanity CORS origins
  - Verify NEXT_PUBLIC_SANITY_APP_ID is set
  - Check Sanity project is accessible
  - Clear browser cache and try again

Content Not Showing:
  - Verify content exists in Sanity Studio
  - Check NEXT_PUBLIC_SANITY_DATASET is 'production'
  - Run 'npm run check:content' locally
  - Check Vercel function logs for errors
